# UPY-01
Добро пожаловать в репозиторий учебного проекта UPY-01 — скромного творения, рожденного в процессе освоения курса по веб-разработке.

##О чём этот проект?
Это полигон для испытаний технологий фронтенда и бэкенда. Здесь мы отрабатывали концепции, а не создавали следующий глобальный сервис.

##Искренняя декларация:
Всё, что вы здесь увидите — чистая фикция. Случайные названия, сгенерированные телефоны и вымышленные адреса существуют лишь для вёрстки и логики кода. Они не несут никакого скрытого смысла и уж тем более не преследуют цели кого-либо задеть.

Технологии и цель — вот что здесь по-настоящему важно.
