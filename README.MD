## Описание проекта
Проект представляет собой сайт, выполненный по заданию, данному на первых двух парах дисциплины "Фронтенд и бэкенд разработка".
За подробным описанием того, что должно присутствовать в проекте, обратитесь к файлу "Практическое занятие №2 Фронтенд и бэкенд разработка (Часть 1 из 2)" из папки "Практические занятия" по данной дисциплине.
